<h3>Breif</h3>
Desktop Java program reads 2D graph images and detect the maximum and minimum point for are graphs in image with x, y values
which on scale in the image. Also it detects the all points for all graphs and get the missing values of graphs which intersected
with another one by using a complex algorithms and mathematics methods to have a small probability of error.
